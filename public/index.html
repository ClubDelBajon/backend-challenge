Morfy - Backend Challenge

Welcome to the Morfy Backend challenge! We are working on a blog backend where users can register, log in and create or read posts.

Your API base url is http://localhost:3000/api/

Please read the README.md carefully before starting.

Ready to start? Here we go:

1. Authorization

There is a security issue: any user can create a post on behalf of another user! Can you fix it?

2. Get Posts

The existing endpoint GET http://localhost:3000/posts retrieves all existing posts. Implement the following functionalities:

  • 2.a) Get posts by author
  • 2.b) Get posts by keyword

3. Comments

Many users are requesting the ability to comment on posts. Each comment should have an author and content.

  • 3.a) Add a new comment to a post
  • 3.b) Include comments when getting a post by ID

4. Tests

We need unit tests for either points 2 or 3 to improve coverage.

5. [Optional] Bonus Track

  • 5.a) The frontend team is asking for API documentation. They suggest Swagger as a good choice. Is it already implemented?
  • 5.b) The infrastructure team requests running the API in a Docker container, pointing to a minified build.


    • Good luck with the challenge! Remember, the goal is to demonstrate your ability to identify and fix issues, implement new features, and write tests. Don't hesitate to ask questions if you get stuck. We're looking forward to seeing your solution!
